📖 Guide: Running the App with Google Sync & Local Autosave

1. Setting up Local Server Auto-saving

  1. Open a terminal inside the project directory: `/Users/amrindersingh/Desktop/Salary tracker/`
  2. Start the local server script: python server.py
  3. Open your browser and navigate to: http://localhost:8080
  4. The server will now automatically save edits back to disk files.

2. Creating Google Cloud Credentials

  1. Go to the Google Cloud Console and create a project.
  2. Under APIs & Services, enable the Google Drive API.
  3. Go to the Credentials tab, and click Create Credentials > OAuth Client ID (Web Application).
  4. Add `http://localhost:8080` to Authorized JavaScript origins.
  5. Copy the generated Client ID and paste it into the Google OAuth Client ID field on this page.
